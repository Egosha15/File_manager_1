# Файловый менеджер на Python

Консольный файловый менеджер с изолированной рабочей директорией. Разработан в рамках лабораторной работы.

## Возможности

- Навигация по файловой системе в пределах рабочей папки
- Создание и удаление директорий
- Создание, чтение, редактирование и удаление файлов
- Копирование, перемещение и переименование файлов
- Защита от выхода за пределы рабочей директории
- Кроссплатформенность (Windows/Linux/macOS)

## Запуск


1. Настроить рабочую директорию в файле config.ini:

work_directory = "Путь к директории"/sandbox

2. Запустить file_manager.py
